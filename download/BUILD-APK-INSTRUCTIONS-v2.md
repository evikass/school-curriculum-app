# Инструкция по сборке APK для RuStore

## Проект: ИнтернетШкола Android

### Описание
React Native приложение с оффлайн-контентом для школьной программы 0-11 классов. Приложение полностью автономно и не требует интернет-соединения.

### Содержание
- 12 классов (подготовительный + 1-11)
- Предметы: Математика, Русский язык, Литература, История, Биология, Физика, Химия, География, Английский, Информатика
- Система XP и достижений
- Мини-игры (квизы)

---

## Метод 1: EAS Cloud Build (Рекомендуется)

### Шаг 1: Подготовка
```bash
# Распакуйте архив
tar -xzvf inetshkola-android-v2.tar.gz -C ./inetshkola-android

# Перейдите в папку проекта
cd inetshkola-android

# Установите зависимости
npm install
```

### Шаг 2: Создание аккаунта Expo
1. Перейдите на https://expo.dev/
2. Создайте бесплатный аккаунт
3. Подтвердите email

### Шаг 3: Сборка APK
```bash
# Войдите в аккаунт Expo
npx eas login

# Запустите сборку APK (preview версия)
npx eas build --platform android --profile preview

# Или production версию для релиза
npx eas build --platform android --profile production
```

### Шаг 4: Скачивание APK
После завершения сборки (10-20 минут):
1. Перейдите на https://expo.dev/accounts/[ваш-логин]/projects/inetshkola-android/builds
2. Скачайте готовый APK файл

## Метод 2: Локальная сборка (требуется Android SDK)

### Требования
- Android Studio с Android SDK
- Java JDK 17+
- Среда разработки настроена для React Native

### Команды
```bash
cd inetshkola-android

# Установите зависимости
npm install

# Сборка через EAS локально
npx eas build --platform android --profile preview --local

# Или экспорт AAB для Google Play
npx expo export --platform android
```

## Информация о приложении

### Пакет
- **Package Name:** `ru.inetshkola.app`
- **Version:** 1.0.0 (versionCode: 2)

### Разрешения (Permissions)
Приложение запрашивает минимальные разрешения:
- Нет интернет-разрешений (полностью оффлайн)
- Нет доступа к файлам
- Нет доступа к камере

### Размер APK
Ожидаемый размер: ~15-25 MB

## Подготовка к публикации в RuStore

### Необходимые материалы
1. **APK файл** - готовый файл из EAS Build
2. **Иконка 512x512** - уже есть в `/download/inetshkola-store-assets/icon-512.png`
3. **Скриншоты** - уже есть в `/download/inetshkola-store-assets/`
4. **Описание приложения:**

```
Название: ИнтернетШкола - Школьная программа оффлайн

Краткое описание:
Полная школьная программа 0-11 класс без интернета!

Полное описание:
ИнтернетШкола - это образовательное приложение для школьников с полным курсом программы 0-11 классов. Работает БЕЗ интернета!

Особенности:
✅ 12 классов (подготовительный + 1-11)
✅ Все основные предметы: математика, русский язык, литература, история, физика, химия, биология, география, английский язык, информатика
✅ Подробные уроки с примерами и заданиями
✅ Система XP и достижений для мотивации
✅ Интерактивные мини-игры и квизы
✅ Работает полностью оффлайн

Идеально подходит для:
- Подготовки к урокам и контрольным
- Повторения пройденного материала
- Изучения новых тем
- Подготовки к ОГЭ и ЕГЭ

Категория: Образование
Целевая аудитория: 6-18 лет
```

### Шаги публикации
1. Войдите в кабинет разработчика RuStore
2. Создайте новое приложение
3. Загрузите APK файл
4. Заполните информацию о приложении
5. Загрузите скриншоты и иконку
6. Отправьте на модерацию

## Технические детали

### Используемые технологии
- **React Native** + **Expo SDK 55**
- **React Navigation** для навигации
- **AsyncStorage** для оффлайн-хранения данных
- **TypeScript** для типизации

### Хранение данных
Все данные пользователя сохраняются локально:
- Выбранный класс
- Прогресс по урокам
- Заработанные XP и достижения
- Настройки приложения

### Оффлайн-режим
Приложение полностью автономно:
- Весь контент встроен в APK
- Нет сетевых запросов
- Нет зависимости от серверов

## Поддержка

При возникновении проблем со сборкой:
1. Проверьте версию Node.js (требуется 18+)
2. Убедитесь, что npm пакеты установлены корректно
3. Проверьте подключение к интернету для EAS Build

Для вопросов по коду: проверьте файлы в папке `src/`
