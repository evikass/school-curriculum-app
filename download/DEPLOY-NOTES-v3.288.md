# ИНЕТШКОЛА v3.288 - Заметки к релизу

## Исправленные проблемы

### 1. SVG-изображения для уроков чтения (1 класс) ✅
- Исправлены пути к SVG файлам - добавлен basePath `/school-curriculum-app`
- Все 20 анимированных SVG теперь доступны

### 2. VK Bridge - локальный пакет ✅
- **Проблема**: VK Bridge загружался с unpkg.com, который может быть заблокирован
- **Решение**: Установлен локальный пакет `@vkontakte/vk-bridge`
- VK Bridge теперь встроен в приложение, а не загружается извне

### 3. Улучшено определение VK окружения
- Добавлена проверка параметра `vk_platform` в URL
- Это поможет правильно определить запуск внутри VK Mini Apps

## Проблемы с VK Mini Apps

Если приложение показывает "ошибка загрузки" или "проверьте подключение":

### Возможные причины:

1. **Кэш VK приложения**
   - Очистите кэш: Настройки VK → Приложения → ИНЕТШКОЛА → Очистить кэш
   - Или удалите и добавьте приложение заново

2. **Настройки VK приложения**
   - URL должен быть: `https://evikass.github.io/school-curriculum-app/`
   - Обязательно со слешем в конце!

3. **Сетевые ограничения**
   - VK может блокировать некоторые запросы
   - Попробуйте другое сетевое подключение

4. **Браузер VK**
   - Обновите приложение VK до последней версии
   - Попробуйте перезапустить приложение VK
